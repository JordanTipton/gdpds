GDP Info Log Format

by Jordan Tipton, 06/05/2016

The following table describes the format of the information log used by the gdp discovery service to read clients' information and store it within the discovery database. Information logs should be a gdp logs.

Record no. Name Description/Value
1 num_capabilities The number of capabilities contained within the info log
2 num_permissions The number of permissions contained within the info log
3 num_pkeys The number of public keys corresponding with clients this info log describes
4 first capability string describing a capability
... capability 2 - n one of n records describing capabilities
variable first permission string describing a permission
... permission 2 - n one of n records describing permissions
variable first public key public key of a client this info log describes in pem format
... public key 2 - n one of n public keys corresponding with clients
variable certificate optional certificate verifying the origin of this information log
